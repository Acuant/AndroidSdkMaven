# AndroidSdkMaven

This repository exists to act as a Maven distribution of the Android SDK. For sample code, docs, and release notes please go to: [https://github.com/Acuant/AndroidSDKV11](https://github.com/Acuant/AndroidSDKV11)

To use this as a maven source for your android project add the following line in your build.gradle under repositories:

    maven { url 'https://raw.githubusercontent.com/Acuant/AndroidSdkMaven/main/maven/' }
